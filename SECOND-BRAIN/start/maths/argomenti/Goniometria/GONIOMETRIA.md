# Angoli associati

$\alpha,\beta$ angoli $\to$ associati se:

$\alpha+\beta=\pi$   e   $\alpha-\beta=\pi$

$\alpha+\beta=\frac{3}{2}\pi$  e  $\alpha-\beta=\frac{3}{2}\pi$

$\alpha+\beta=\frac{\pi}{2}$   e   $\alpha-\beta=\frac{\pi}{2}$

$\alpha+\beta=2\pi$   e   $\alpha-\beta=2\pi$

# Formule di addizione e sottrazione

$$sen(\alpha+\beta)=sen\alpha \ cos\beta+cos\alpha \ sen\beta$$
$$sen(\alpha-\beta)=sen\alpha \ cos\beta-cos\alpha \ sen\beta$$
$$cos(\alpha+\beta)=cos\alpha \ cos\beta-sen\alpha \ sen\beta$$
$$cos(\alpha-\beta)=cos\alpha \ cos\beta+sen\alpha \ sen\beta$$


$$tg(\alpha+\beta)=\frac{sen(\alpha+\beta)}{cos(\alpha+\beta)}=\frac{sen\alpha \ cos\beta+cos\alpha \ sen\beta}{cos\alpha \ cos\beta-sen\alpha \ sen\beta}=\frac{tg\alpha+tg\beta}{1-tg\alpha-tg\beta}$$

$$ctg(\alpha+\beta)=\frac{ctg\alpha \ ctg\beta-1}{ctg\alpha+ctg\beta}$$

# Formule di duplicazione

$$sen2\alpha=2sen\alpha \ cos\alpha$$
$$cos2\alpha=cos^2\alpha-sen^2\alpha   \quad\text{o}\quad   cos2\alpha=1-2sen^2\alpha \quad\text{o}\quad cos2\alpha=2cos^2\alpha-1$$
$$sen^2\alpha=2cos^2\alpha-1$$
$$cos^2\alpha+sen^2\alpha=1$$

$$sen^2x=1-cos^2x$$
$$cos^2x=1-sen^2x$$
$$-sen\alpha=cos^2x-1$$
$$tg(2\alpha)=\frac{2sen\alpha \ cos\alpha}{cos^2\alpha \ sen^2\alpha}=\frac{2tg\alpha}{1-tg^2\alpha}$$
$$ctg2\alpha=\frac{cos^2\alpha}{sen2\alpha}=\frac{cos^2\alpha-sen^2\alpha}{2sen\alpha \ cos\alpha}=\frac{ctg^2\alpha-1}{2ctg\alpha}$$

# Formule di bisezione

$$sen\frac{\alpha}{2}=\sqrt{\frac{1-cos\alpha}{2}}$$
$$cos\frac{\alpha}{2}=\sqrt{\frac{1+cos\alpha}{2}}$$
$$tg\frac{\alpha}{2}=\sqrt{\frac{1-cos\alpha}{1+cos\alpha}}=\frac{1-cos\alpha}{sen\alpha}=\frac{sen\alpha}{1+cos\alpha}$$
$$1-cos^2\alpha=(1+cos\alpha)(1-cos\alpha)$$
