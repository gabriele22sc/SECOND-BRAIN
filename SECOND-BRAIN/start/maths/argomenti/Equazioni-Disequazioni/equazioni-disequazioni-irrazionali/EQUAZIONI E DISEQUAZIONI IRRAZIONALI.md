
# Equazione irrazionale
## Definizione

Un'equazione o disequazione è irrazionale se in essa ci sono radicali contenenti l'incognita.


## Se n è dispari 

$\sqrt[n]{A(x)}=B(x)$  è equivalente ad $A(x)=[B(x)]^2$  se n è dispari


## Se n è pari

In sintesi, se n è pari, l'equazione $\sqrt[n]{A(x)}=B(x)$  è equivalente al sistema:

$\begin{cases} A(x)\geq0\\ B(x)\geq0\\ A(x)=[B(x)]^2 \end{cases}$







Esistono due modi per risolverle:

# Sostituzione del risultato all'equazione

$\sqrt{3x+4}=x$ 

elevo al quadrato entrambi i membri $(\sqrt{3x+4})^2=(x)^2 \quad\rightarrow\quad 3x+4=x^2$ 

risolvo l'equazione ottenuta $x^2-3x-4=0 \quad\rightarrow\quad x_{1,2}=\frac{3\pm\sqrt{25}}{2}$   $x_1=4$  e $x_2=-1$

verifico se le soluzioni ottenute vanno bene sostituendo nell'equazione di partenza

$\sqrt{3*4+4}=4 \quad\rightarrow\quad \sqrt{16}=4 \quad\rightarrow\quad 4=4$   si, quindi 4 è soluzione

$\sqrt{3*(-1)+4}=-1 \quad\rightarrow\quad \sqrt{1}=-1 \quad\rightarrow\quad 1=-1$  no, quindi -1 non è soluzione 

# Ponendo entrambi i membri >=0

se voglio che sia uguale ad un radicale di indice pari anche $x$ deve essere $\geq0$ quindi impongo:

$x\geq0$ 

quindi $4$ sarà soluzione mentre $-1$ non sarà soluzione



# Disequazione irrazionale

## Se n è dispari

$\sqrt[n]{A(x)}<B(x)$  è equivalente a  $A(x)<[B(x)]^n$;
$\sqrt[n]{A(x)}<B(x)$  è equivalente a  $A(x)>[B(x)]^n$.

## Se n è pari

Se a e b sono due numeri reali positivi o nulli, la relazione di disuguaglianza che c'è fra i dure numeri è la stessa che c' fra i loro quadrati:

la disequazione $\sqrt{A(x)}<B(x)$  è equivalente al sistema:
$\begin{cases}A(x)\geq0\\ B(x)>0\\ A(x)<[B(x)]^2\end{cases}$

inoltre abbiamo due casistiche:

# $\sqrt[n]{A(x)}<B(x)$

- $A(x)<[B(x)]^n$  se n è dispari;

- $\begin{cases} A(x)\geq0\\ B(x)>0\\ A(x)<[B(x)]^n\end{cases}$


# $\sqrt[n]{A(x)}>B(x)$

- $A(x)>[B(x)]^n$  se n è dispari;

- $\begin{cases} B(x)<0\\ A(x)\geq0 \end{cases}$    $\cup$    $\begin{cases} B(x)\geq0\\ A(x)>[B(x)]^n\end{cases}$   se n è pari
