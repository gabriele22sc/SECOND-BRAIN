È un metodo matematico utilizzato per creare calendari di tornei sportivi in formato girone all’italiana (Round-Robin), assicurando che ogni partecipante affronti tutti gli altri una volta sola in $n-1$ giornate(per n squadre pari). Divide le squadre in due metà, accoppiandole e facendole ruotare per generare i turni.

## Funzionamento
- **Numero pari**: con n squadre, si giocano $n-1$ giornate, ciascuna con $n/2$ incontri;
- **Rotazione**: per generare le giornate, una squadra resta Fissa, mentre le altre ruotano in senso orario o antiorario.
- **Numero dispari**: Se il numero di squadre è dispari, si aggiunge una squadra “fittizia”: chi affronta questa squadra nel turno specifico non gioca.
