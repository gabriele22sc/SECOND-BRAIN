Esistono due tipi di software all’interno di un PC:
- **Software di base**: programmi per l’utilizzo e la manutenzione del computer;
- **Software applicativo**: programmi per l’utilizzo specifico del computer in un determinato campo.

# Software di base
Riunisce tutti gli strumenti che consentono al computer di operare, quello per eccellenza è il **sistema operativo**. Serve per poter gestire e mettere a disposizione dell’utente l’insieme delle risorse di cui si compone l’hardware.
I suoi compiti sono:
- Controllo del corretto funzionamento di ogni componente;
- Gestione e memorizzazione delle informazioni;
- Gestione errori;
- Gestione vie di comunicazione nelle reti;
- Supporto software applicativo compatibile.

## All’accensione del PC
1. La CPU esegue il POST(Power-On Self Test): 

Le istruzioni della CPU vengono impartite da una parte del software di sistema chiamata **BIOS**(Basic Input Output System) che riguardi nella **ROM**(Read Only Memory) del PC. Durante questa sequenza di istruzioni la CPU effettua una serie di controlli sul PC:
- Quanta memoria RAM ha a disposizione e se funziona;
- Tastiera e mouse collegati;
- Il tipo di BUS di cui si dispone;
- Dispositivi di memoria di massa(dischi).
Se durante questa fase di test vengono rilevati errori(malfunzionamenti) la CPU stampa a monitor i messaggi di errore ed emette alcuni suoni per avvisare dell’esistenza di problemi nella sequenza di POST.

2. Successivamente, **Bootstrap**, ovvero la sequenza di caricamento:
Se la sequenza di POST termina senza errori, il BIOS procede con la sequenza di caricamento detta **bootstrap**. Questa operazione carica nella RAM il sistema operativo.

# Sistema Operativo
A seconda di come il SO dialoga con l‘utente(interfaccia utente), esso può essere:
- **Command line**: l’interfaccia è scarna e l’utente deve conoscere i comandi permessi e la loro sintassi (es. MS-DOS);
- **Graphic User Interface(GUI)**: tutti i programmi e le funzioni sono mostrate sullo schermo mediante simboli immediatamente riconoscibili, l’approccio è intuitivo e facile da gestire(es. Windows, Linux, Mac-OS).

## Definizione
Un sistema operativo(SO) è un programma che controlla l’esecuzione di programmi applicativi e agisce come interfaccia tra le applicazioni e l’hardware del calcolatore

## Obiettivo
- **Efficienza**: un SO cerca di utilizzare in modo efficiente le risorse del calcolatore
- **Semplicità**: un SO dovrebbe semplificare l’utilizzazione dell’hardware di un calcolatore.

## Componenti:
- **Gestione dei processi**;
- **Gestione della memoria principale**;
- **Gestione della memoria secondaria**;
- **Gestione file system**;
- **Gestione dei dispositivi di I/O**;
- **Protezione**;
- **Networking**;
- **Interruttore dei comandi**;

# Software applicativo
Categoria che raggruppa le applicazioni d’interesse generale ed i programmi più settoriali.
Qui rientra anche il software utilizzato nell’attivazione del lavoro di ufficio:
- Elaborazione di testi;
- Fogli di calcolo;
- Database;
- Programmi di presentazione;
