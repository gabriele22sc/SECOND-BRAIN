# Un pò di Storia

#Ngrok è stato fondato con l'obiettivo **"dare potere agli sviluppatori"** e di semplificare lo sviluppo in locale, offrendo un comando semplice per esporre le applicazioni in esecuzione in locale al web: `ngrok http 80`.

# [Come nasce e qu<mark style="background: #FF5582A6;"></mark>ali sono gli obiettivi di Ngrok](https://ngrok.com/next-generation?utm_source=chatgpt.com)

Nel corso del tempo, ngrok è diventato una piattaforma di ingresso unificata, combinando componenti come reverse proxy, load balancer, API gateway, firewall e protezione DDoS in un'unica soluzione. Questa evoluzione ha permesso a ngrok di essere utilizzato in una vasta gamma di applicazioni, dalle installazioni artistiche ai sistemi HVAC, dai dispositivi IoT alle transazioni finanziarie critiche in banche, ospedali e governi.
