
#REM	->	"root-em", la sua dimensione riferisce alla dimensione del font dell'elemento radice, ad esempio html. La dimensione è calcolata in base alla dimensione del font dell'elemento, impostata come valore di base; se il font-size dell'elemento html è 16px, 1rem sarà uguale a 16px, 2rem sarà uguale a 32px, e così via.

#EM	->	"Relative to parent element", relativo al font-size dell'elemento che lo utilizza o relativo al font-size del suo elemento genitore. Se un elemento definito in em, la sua dimensione si moltiplicherà per la dimensione del font dell'elemento di riferimento. Viene utilizzato per adattare dinamicamente un elemento alla dimensione del font del suo contesto.

#VH	->	"Viewport Height", si basa sull'altezza della view. 1vh equivale all'1% dell'altezza.

#VW	->	"Viewport Width", si basa sulla larghezza della view. 1vw equivale all'1% della larghezza.

#VMIN	->	"Viewport minimum", si basa sulla dimensione più piccola della view.

#VMIN	->	"Viewport maximum", si basa sulla dimensione più larga della view.
