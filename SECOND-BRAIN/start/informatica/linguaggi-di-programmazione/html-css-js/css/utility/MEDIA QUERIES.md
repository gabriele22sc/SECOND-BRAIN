Vengono usate per definire gli stili di determinate classi per diverse grandezze di schermo(mobile, pc, monitor…)

[media queries](https://www.w3schools.com/css/css3_mediaqueries_ex.asp)
 es:
 
```
 /* Media query per schermi più piccoli da cambiare background*/
@media (max-width: 767px) {}

/* media query per schermi più grandi (min-width: 768px) */
@media (min-width: 768px) {
```
