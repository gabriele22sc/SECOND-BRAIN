
## PILLOW

#Pillow è una libreria per Python che permette di gestire e manipolare immagini. È una versione aggiornata e mantenuta attivamente della storica libreria PIL (Python Imaging Library). Con Pillow puoi:
- Aprire e salvare immagini in diversi formati (come PNG, JPEG, GIF, ecc.).
- Modificare le immagini, ad esempio ritagliandole, ridimensionandole, ruotandole o applicando filtri e trasformazioni.
- Disegnare su immagini, aggiungendo testo, forme, linee o altri elementi grafici.
- Creare immagini da zero, come nel caso dello script per generare uno sfondo stellato.

In sostanza, Pillow è uno strumento molto utile per chi lavora con la grafica o ha bisogno di elaborare immagini direttamente tramite codice Python.
Per verificare che sia installato:
pip show Pillow

[REFERENCE](https://pypi.org/project/pillow/)
