# Per cambiare il daily message
1. digita `vi etc\motd`
2. digita `I` per la insert mode
3. cambia il messaggio
4. digita ESC
5. digita `:wq` per salvare le modifiche
6. chiudi e riapri e il gioco è fatto

# Montare cartelle
digita:
`mount -t ios . /nome_cartella`
**NB**: è consigliato metterle sempre dentro la cartella `/mnt` o creare una cartella apposita per i file
